GrizzlySpark

GrizzlySpark es una plataforma web en línea especialmente diseñada para la creación, alojamiento y gestión de desafíos CTF (Capture The Flag) enfocados en ciberseguridad y hacking ético. La plataforma permite a los usuarios tanto crear sus propios retos personalizados como participar en desafíos ya existentes, facilitando el aprendizaje práctico y la mejora de habilidades en áreas clave de la seguridad informática.

.NET C# Blazor ASP.NET Javascript HTML CSS

Clon Microsoft Learn

Este proyecto se centra exclusivamente en replicar la interfaz gráfica y la experiencia de usuario de Microsoft Learn, con el objetivo de ofrecer una experiencia visual e interactiva similar a la plataforma oficial de aprendizaje. La iniciativa busca recrear de forma precisa los elementos de diseño, la disposición de los contenidos, la navegación intuitiva y la interactividad característica de Microsoft Learn.
